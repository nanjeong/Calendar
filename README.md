# Calendar

## 구현할 것

- 상단에 날짜 표기

  - 오늘 요일 표기
  - 오늘 날짜 표기
  - 오늘 월 표기
  - 오늘 연도 표기

- 달력처럼 보이도록 하기

  - 일, 월, 화, 수, 목, 금, 토 라벨링
  - 현재 월의 1일이 무슨 요일인지 판별
    - 해당 요일 라벨 아래 1일 표기
  - 월의 마지막 날까지 달력에 표기

- 화살표 클릭 시

  - 우측 화살표 클릭 시 다음 달 달력 보여주기
  - 좌측 화살표 클릭 시 이전 달 달력 보여주기

- 특정 날짜를 클릭했을 때, 상단에 요일 및 날짜 반영하기
